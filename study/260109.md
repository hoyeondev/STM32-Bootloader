# UART Driver & RealTerm Debugging (STM32)

### 1. UART(Universal Asynchronous Receiver/Transmitter)
UART는 비동기 직렬 통신 방식으로, MCU와 외부 장치(PC, 센서, 모듈 등) 간 데이터를 송수신하는 데 사용된다.

**UART 특징**
- 클럭 신호 없이 통신 (Asynchronous)
- 송신(TX), 수신(RX) 핀 사용


### 2. RealTerm
RealTerm은 Windows 환경에서 사용되는 고급 시리얼 통신 터미널 프로그램으로,
UART / RS232 / RS485 기반의 임베디드 시스템과 PC 간 데이터 송수신 및 디버깅을 위해 사용된다.

일반적인 터미널 프로그램(PuTTY, TeraTerm 등)보다
바이너리 데이터 처리와 저수준 통신 테스트에 특화되어 있다.

- [다운로드 링크](https://sourceforge.net/projects/realterm/)
